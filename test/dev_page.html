Update Scanner Development Environment

Setup

  • Download the latest source code from [TBD]
  • Create a new Firefox profile called "updatescanner_dev"
  • Install the Extension Auto-Installer addon
  • Close Firefox

Usage

  • Type the following to launch the Firefox dev profile: 
        make dev
  • Type the following in different terminal to automatically post XPI updates to the Firefox dev profile: 
        make watchpost
