# Update Scanner
Monitors webpages for updates

# Development
You'll need:
* [Node.js](https://nodejs.org/)
* [npm](https://www.npmjs.com/)

Clone the source and install all dependencies:

    git clone <<insert_repo_here>>
    cd updatescanner
    npm install
    npm install grunt-cli --global
